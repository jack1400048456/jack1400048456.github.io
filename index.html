js基础

require和import的区别

两者的加载方式不同，规范不同，require是在运行时加载，而import是在编译时加载，require（./a)()此时导入的模块是一个函数，类比即也可导入一个对象或一个数组。import * from "./a"一般用在开头。
并且规范不同，require是Commonjs规范，import是es6规范

对于闭包的理解？优缺点？

概念：闭包就是能够读取其他函数内部变量的函数

特性：

  1. 函数嵌套函数
  2. 函数内部可以引用外部的参数和变量
  3. 参数以及变量不会被垃圾回收机制回收

优点：

  1. 希望一个变量长期储存在内存当中
  2. 避免全局变量的污染

缺点：

  1. 使用不当易造成内存泄露
  2. 常驻内存，增加内存使用量

javascript 的 typeof 返回哪些数据类型

string number boolea objec undefin function 以及es6新增数据类型symbol

一次 js 请求一般情况下有哪些地方会有缓存处理

cnd缓存处理 dns缓存处理，浏览器缓存，服务器缓存。

js 常见数组方法

  1. splice，会直接与数组进行修改 array.splice(index,howmany,item1) index(必须)添加/删除的位置，howmany(必须)删除的数量，数量为0则不删除，item1要添加的新项目
  2. slice，返回一个指定的数组，array.slice(start,end)，start(必须)如果为负数，则从数组尾部开始算起，-1指向最后一个元素，end(如果没有这个参数，则从start到最后)
  3. join，将数组中的所有元素放入一个字符串，array.join(separator)separator为所拼接的分隔符，如省略则默认使用逗号
  4. split，用于将一个字符串分割成一个字符串数组string.split(separator,howmany),separator(为字符串或者正则表达式，从该参数指定的地方开始分割),howmany(可选，返回的子串不会超过指定的数组，如没有指定则整个数组都会被分割)
  5. filter，创建一个新的数组，新数组按照检测指定函数符合条件的所有元素。array.filter(function(Value,index,arr))，value(必须)当前元素的值，index当前元素的索引值，arr当前元素的数组对象。
  6. push，向数组的末尾添加一个或更多个元素
  7. unshift，向数组的开头添加一个或更多个元素
  8. shift，删除并且返回数组的第一个元素
  9. pop，删除并且返回数组的最后一个元素

apply() call() bind()

这三个函数均是用来指定this指向的对象，比如a.call(b)就是将a对象放到b的环境中使用，三者均可以如此使用，并且均可带多个参数，而三者不同点则是apply()后传入的必须是数组a.apply(b,[])。bind()返回的则是一个函数，必须继续调用a.bind(b)()。

//如果在call() apply()中传入的第一个参数是null则this指向的对象是window

this的指定向问题

对于this的指向问题，首先要说的是this的指向是不确定的，当某个对象调用它的时候，那么this就会指向调用它的对象。比如

在这里可以表明如果一个函数中有this，这个函数有被上一级的对象所调用，那么this指向的就是上一级的对象

并且如果一个函数中有this，这个函数中包含多个对象，尽管这个函数是被最外层的对象所调用，this指向的也只是它上一级的对象

而在实际中，如果在一个函数中又this，但是它没有被上一级的对象所调用，那么这个this就是指向的window
